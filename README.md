## 📖 About the Project
**EcoAl** is a strategic corporate platform developed as part of the **FIAP** educational ecosystem. The project addresses the growing market demand for transparency and efficiency in **ESG** (*Environmental, Social, and Governance*) management.

Unlike generic monitoring tools, EcoAl was designed specifically for the corporate environment, offering a centralized hub to track sustainability metrics, social responsibility, and governance compliance.

### 🎯 Core Objectives
* **Indicator Management:** Centralizing critical data for socio-environmental impact-driven decision-making.
* **Goal Monitoring:** A dedicated tracking system for corporate milestones, allowing companies to visualize progress toward sustainable targets.
* **Gamified Engagement:** Using gamification logic to encourage employees and stakeholders to hit ESG milestones.
* **Transparency Culture:** Streamlining data reporting for stakeholders, investors, and regulatory bodies.

## 🏗️ Architecture & Technical Highlights

To ensure a robust and scalable solution, the app follows modern Android development standards:

* **Seamless UX:** Authentication flows implemented via **Bottom Sheets**, keeping the user engaged without abrupt screen transitions.
* **Clean Architecture Patterns:** Clear separation of concerns between UI Components, Screens, and State Management.
* **Visual Polish:** Implementation of **Dynamic Blur** (Glassmorphism) and sequential animations to enhance the "Premium" feel.
* **UDF (Unidirectional Data Flow):** Ensuring predictable state and easier debugging.

---

## ✅ Features Implemented

### 1. Advanced Splash Screen
- **Sequential Animations:** Logo scaling and positioning transitions.
- **Visual Effects:** Dynamic background blur (8.dp radius) and glassmorphism.
- **Smooth Transitions:** Content fade-in using Jetpack Compose animation states.

### 2. Modern Authentication System (Modals)
- **Bottom Sheet Integration:** Login and Sign-up forms that slide over the main context.
- **Dynamic Form Validation:** Real-time state management for input fields.
- **Password Visibility Toggle:** Custom trailing icons with reactive UI updates.
- **Responsive Design:** Fully adapted for different screen sizes with scrollable containers.

## 🚀 How to Run

1.  Clone the repository:
    ```bash
    git clone [https://github.com/pmiranda27/esg-ecoal.git](https://github.com/pmiranda27/esg-ecoal.git)
    ```
2.  Open the project in **Android Studio (Ladybug or newer)**.
3.  Sync project with **Gradle Files**.
4.  Run on an emulator or physical device with **API 24+**.
